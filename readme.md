## Overview

Generates a static info-page about the world, stats include:

* Available nodes and their properties
* Available tools
* Available items
* Recipes
* Active ABM's
* Active LBM's

## Usage

* Install the mod
* Start the world
* Open the generated `index.html` in the world-folder `mtinfo` with a web-browser
